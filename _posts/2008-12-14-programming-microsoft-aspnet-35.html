Having heared only positive things about Dino Esposito</a>'s way of writing i decided to read Programming Microsoft ASP.NET 3.5</a>. The book is written in a way that invites you to read and experiment with the contents. It covered the essentials of ASP.NET programming but i did not have the feeling that it added much to what i had read in Professional ASP.NET 2.0</a>. After a while i became annoyed with the numerous references to Programming Microsoft ASP.NET 2.0 Applications: Advanced Topics</a>.</p>
