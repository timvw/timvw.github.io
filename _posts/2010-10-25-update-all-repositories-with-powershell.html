I typically store the repositories i am working on under D:\Code. Each morning i had to right click on each of those folders and select 'SVN Update' using Tortoise SVN</a>. Today i decided there had to be a better way to accomplish this tedious task:</p>

[code lang="powershell"]
dir d:\code | foreach { svn update $_.FullName }
[/code]

And in case you really like tortoise, you can do the following:</p>

[code lang="powershell"]
dir c:\code | foreach { tortoiseproc /command:update /closeonend:1 /path:$($_.FullName) }
[/code]
