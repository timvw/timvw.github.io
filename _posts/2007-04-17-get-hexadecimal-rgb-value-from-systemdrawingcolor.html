Here's a simple function that returns the hexadecimal</a> RGB</a> value of a System.Drawing.Color</a>:</p>
[code lang="csharp"]private string ToHexadecimalRgb(Color color)
{
return color.ToArgb().ToString("X").Substring(2);
}[/code]
