In order to debug an MSBuild</a> script i added a couple of tasks, but when i asked Visual Studio to Build i didn't get to see the output... By default Visual Studio will use "Minimal" as verbosity</a> level. You can change this via Tools -> Options -> Projects and Solutions -> Build and Run.</p>
