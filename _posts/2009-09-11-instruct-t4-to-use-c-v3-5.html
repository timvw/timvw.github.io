Consider this simple T4 template:</p>

[code lang="xml"]<# for (var i = 0; i < 10; ++i) { WriteLine("hello"); } #>[/code]

Trying to build the project results in a compilation error because 'var' is an unknown type. A bit of research learned me that i should instruct the processor to use a specific c# version like this:</p>

[code lang="xml"]<#@ template language="C#v3.5" inherits="Microsoft.VisualStudio.TextTemplating.VSHost.ModelingTextTransformation" #>
<# for (var i = 0; i < 10; ++i) { WriteLine("hello"); } #>[/code]

Problem solved :)</p>
