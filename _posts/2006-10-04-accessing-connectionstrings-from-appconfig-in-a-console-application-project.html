Earlier today i added a configuration file to my Console Applicaton Project (Add Item -> Application Configuration File). Via ConfigurationSettings</a> i could only access the AppSettings. It took me a while to figure out that i had to add a reference to System.Configuration.dll. Once that was done i could access the ConnectionStrings via ConfigurationManager</a>.</p>
