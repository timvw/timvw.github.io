Earlier this week someone asked me how he could select custom objects from a ComboBox</a>. Here is the code he used:</p>
[code lang="csharp"]private void FillComboBoxPersons(List persons) {
this.comboBoxPersons.Items.Clear();
this.comboBoxPersons.Items.Add( "--- Select Person -------------------" );
foreach ( Person person in persons ) {
this.comboBoxPersons.Items.Add( person.Name );
}
this.comboBoxPersons.SelectedIndex = 0;
}[/code]

In order to get the selected item he then used the SelectedIndex</a> property to lookup the Person in a cache of the persons collection.</p>

Here is an approach that doesn't require you to have a cache of the collection (Since the persons are already stored in the items):</p>
[code lang="csharp"]private void FillComboBoxPersons(List persons) {
this.comboBoxPersons.Items.Clear();
this.comboBoxPersons.Items.Add( "--- Select Person -------------------" );
this.comboBoxPersons.DisplayMember = "Name";
foreach ( Person person in persons ) {
this.comboBoxPersons.Items.Add( person );
}
this.comboBoxPersons.SelectedIndex = 0;
}[/code]

Now you can easily access the selected item through the SelectedValue</a> property.</p>
