Here are some tips that i want to share with fellow .Net developers that use git on cygwin.</p>

First of all i defined some aliases in my ~/.bashrc:</p>

[code lang="bash"]
# open explorer in the current working directory
alias explorer='explorer.exe "`cygpath -aw \"$PWD\"`"'

# invoke MSBuild
alias msbuild='/cygdrive/c/Windows/Microsoft.NET/Framework/v4.0.30319/MSBuild.exe'
[/code]

Because i don't like the TFS source control story i use git-tfs</a>. As a .Net developer you want to add the following to your .git/info/exclude file:</p>

Whenever i work online i usually run these two commands consecutively: git -a -m "commit message" and git-tfs checkin -m "commit message". Here is a small ~/bin/commit script that combines these:</p>

[code lang="bash"]
#!/bin/bash
git commit -a -m "$1";
git-tfs checkin -m "$1";
[/code]
