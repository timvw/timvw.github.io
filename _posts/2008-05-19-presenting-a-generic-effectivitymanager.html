I've already presented a Generic Effectivity</a>. Offcourse, managing all these effectivities (versions of data) can be handled in a generic way too. A bit of experience mixed with Patterns for things that change with time</a> allowed me to come up with the following interface:</p>
[code lang="csharp"]public interface IEffectivityManager : IList>
{
IEffectivity Add(T t, DateTime begin);
IEffectivity GetSnapshot(DateTime validityDate);
bool TryGetSnapshot(DateTime validityDate, out IEffectivity effectivity);
}[/code]
