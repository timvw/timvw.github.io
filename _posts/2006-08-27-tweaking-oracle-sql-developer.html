A couple of days ago i discovered Oracle SQL Developer</a>, a new and free graphical tool for database development. At first i was impressed by all it's features but when i tried to modify a couple of existing stored procedures the application freezed. EricH</a> directed me to the FAQ: Can I suppress Code Insight (and why would I want to)?</a>. Now that i have added the 'AddVMOption -J-Dsdev.insight=false' to my sqldeveloper.conf the application runs smooth</p>
