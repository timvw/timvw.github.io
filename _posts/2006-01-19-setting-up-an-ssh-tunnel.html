On the machine example there is a (tcp/ip</a>) program listening on port 12345. The protocol it talks is some plaintext</a> language. I want to talk with it, but i don't want others to know what i'm sending to it. I'm lucky enough to have remote access to that machine via ssh</a>. I setup a tunnel with the following command:</p>
[code lang="bash"]
ssh -N -L 12345:example:12345 timvw@example
[/code]

Now my program can connect to localhost:12345 and ssh will make sure that it ends up at example.:12345 without others being able to see the actual data :) For windows users i suggest that you take a look at Plink</a>.</p>
