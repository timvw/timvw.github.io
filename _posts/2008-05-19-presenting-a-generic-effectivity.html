Very often we have to manage objects and their changes over time. Usually we implement this by adding a Range</a> to the data. Martin Fowler</a> has a name for this pattern: Effectivity</a> and i have an implementation for the following interface:</p>
[code lang="csharp"]public interface IEffectivity : IComparable>
{
T Element { get; }
IRange ValidityPeriod { get; }
bool IsEffectiveOn(DateTime validityDate);
}[/code]

Feel free to download IEffectivity.txt</a>, Effectivity.txt</a> and EffectivityTester.txt</a>.</p>
